
## Name 

Boleta Eléctrónica

## Requirements 

- Versión Node: 18.20.7
- Run dev : npm run dev
- Run prod : npm run start

## Description
App web que permite 
- Registrar una venta de auto partes 
- ver reportes por fecha 
- descargar en pdf y exel los registros de vetas
- filtrar boletas por fecha, dni, ruc.

## installation
- npm install
